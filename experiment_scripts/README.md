# Scripts for running experiments, mainly parameter space searches.

- archive/ contains old experiment scripts which have been finished with

wco\_parameter\_search.py - code for searching the parameter space, primarily varying c values and noise values.

wco\_exp\_3.py - code for searching the parameter space, varying c values and initial frequency values.
               - 5 cores

wco\_exp\_4.py - code for searching the parameter space, varying c values and pulse amplitude values.
               - 5 cores

wco\_exp\_5.py - code for searching the parameter space, varying c values and pulse width values.
               - 5 cores

wco\_exp\_6.py - code for searching the parameter space, varying c values and initial phase values.
               - 6 cores

wco\_exp\_7.py - code for searching the parameter space, varying c values and noise values, allowing for two different noise values for phase and freq.
               - 25 cores

wco\_exp\_8.py - code for searching the parameter space, varying c values and initial phase values.
               - 8 cores
