# Temporary and permanent data analysis files for teh tapping data. Will try to keep some organisation.

## Files:

- all\_conditions\_granger.py: code to run granger causality tests on any/all experiment conditions.

- correlations.py: Looking at the correlation between an individuals ie times when they are tapping alone vs
  when they are tapping as a leader.

- data\_summaries.py: summarises the correlations in the empirical data/the mean frequencies and ie times, etc.

- frequency\_distributions.py: comparing stats of participant 1 to participant 2 (poorly named).

- get\_condition\_plots.py: getting histograms of frequencies by the different conditions.

- get\_distribution\_plots.py: For each condition, plot the ie times of each participant to see how they vary. 
  This and previous file's plots are in write-up document.

- initial\_frequencies.py: Looking at the frequency data during the first 5 seconds of data. This is to estimate
  the initial frequency of participants, before something like 'joint rushing' has time to occur.

- tap\_times.py: Looking at the raw tap times, rather than ie times, to try and see if one starts tapping after another
  with the motivation of trying to get an initial phase condition.
